# tool-report-app
Разработка печатной формы отчета о применении режущего инструмента
Ссылка на сайт: https://kriskoshcheeva.github.io/tool-report-app/

Возможности:
- выбор инструмента из 15 позиций
- генерация паспорта инструмента формата А4
- штрихкод с индивидуальным номером
- история выдачи с поиском
- возможность распечать паспорт повторно
- очищение истории
